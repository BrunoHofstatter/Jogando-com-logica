# Teacher's Manual - Pedagogical Context

## 🎓 Core Educational Objective
The "Jogando com Lógica" platform serves as an interactive auxiliary tool for mathematics and logic educators. The games are specifically tailored to let students practice arithmetic, spatial planning, working memory, and rapid problem-solving while feeling completely immersed in play, rather than taking a traditional "test".

## 🏫 Classroom Application
- **Zero Friction:** Teachers do not need to register their students, create classes, or manage passwords. The students simple open the link and start playing immediately.
- **Format:** The platform is versatile. It can be used individually on mobile phones/tablets or collaboratively in pairs on school computers. 
- **Timeframe:** Perfect for filling the standard 40-50 minute school lab sessions with engaging, productive activities.

## 🧠 Targeted Competencies
1. **Mental Math (Aritmética Mental):** Rapid calculation under pressure (e.g., *Stop Matemático*, *Caça Soma*).
2. **Spatial Logic (Lógica Espacial):** Thinking ahead, predicting opponent moves, and understanding grid relationships (e.g., *Crown Chase*, *Math War*, *Damas*).
3. **Problem Solving (Resolução de Problemas):** Breaking down complex situations into logical steps (e.g., *Rubik's Class*).
4. **Resilience:** Learning to lose, iterate, and try different strategies without frustration.
